SAVE A FILE IN MICROSOFT WORD:

Saving a file is critical for editing,preserving and sharing your work.If the programs closes or your computer shut down unexpectedely while you are working on your microsoft word don't worry,by defualt word automically saves your work every minutes and will reload that save upon reopening the document.However,it is still a good idea to save your work per minute or any how you wish to on your microsoft word.

To save a file in microsoft word you can click the save icon on the top-left corner,then click file > save, or use the shortcut Ctrl + s(command + s for ) click the save icon on above.First you must choose where you want to save your documents either on One Drive, Computer , Add a Place. If you save a lot of docunent to a particular folder,you can add that folder as a Save As option

BROWSE :

To save in a different folder not listed above,click Browse,this will take you to a dialogue box that allows you to search your computer to find the folder which you wish to store or save your document.Once you are in the save dialogue box,you can choose the folder you want to save your file in.Select the folder,then type a name for your file in the File name field.Ensure that you are saving the document as the correct file type.There are variety of option to choose,but you will always chooseWord Document(which is the default),then click save in the left bottom corner next to the cancel botton.Congratulations!your file is saved,as you continue your work please save periodically with Ctrl + S.
