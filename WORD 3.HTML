Kindly, watch the short video below on how to create a new document in word or follow the step below.

Try it!

To create a document, simply open Word, select a blank document or template, and start typing. Word offers many professionally designed templates to help you create letters, resumes, reports, and more.

Create a blank document

  1. Double click on microsoft word icon on your desktop
  2. Open Word. Or, if Word is already open.
  3. Click on the file tab.
  4. Click on the new tab.
  5. Under the available templates click on the blank document.

Create a document using a template

  1. Open Word. Or, if Word is already open, select File and then click on New.

  2. Double-click a template to open it.
