• HOME TABS:

    • This is the most used tabs, it incorporates all text formatting features such as font and paragraph changes.

  • INSERT TABS:

    • This tabs allows you to inter a variety of items into a document from pictures,clip arts,tables and header and footers.

  • PAGE LAYOUT TAB:

    • This tab has the command to adjust page elements such as margins,orientation,inserting coiumns,page background and themes.

  • REFERENCE TAB:

    • This tab is used when creating a table of content and citation page for a paper.it provides you with many simple solution to create typically difficult to produce documents.
  • MAILING TAB:

    • This allows you to create document to help when sending out mails such as printing envelopes,labels and processing mail merges.
  • REVIEW TAB:

    • This tab allows you to make any changes to you document due to spelling and grammer issues. it also hold the track changes features which provide people with the ability to make notes and changes to document of other people.
  • VIEW TAB:

    • This tab allows you to change the view of your document to a different different two page document or zoom.
